# Clash Royale Quartett

Ein Quartett-Kartenspiel mit Clash Royale Charakteren.

## Schnellstart

```bash
# Dependencies installieren
bundle install

# Bilder vom Wiki herunterladen
ruby clashroyale_bilderdownload_und_konvertierung_neues_design.rb

# PDF generieren
ruby kartenerstellung_neues_Design.rb
```
